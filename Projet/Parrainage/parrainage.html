Suñu Parrainage

Application web de gestion des parrainages pour les élections présidentielles au Sénégal

Code Source Voir le Projet

Présentation du Projet

Dans le cadre du processus électoral sénégalais, les candidats doivent obtenir un certain nombre de parrainages. J'ai développé avec des camarades de classe M.FALL, M.M.J.SYLLA, P.Y.MBODJ et H.S.SENE une application web permettant de centraliser et de gérer efficacement ces parrainages.

Fonctionnalités Principales

Utilisateur DGE

  • Définition de la période de parrainage par le DGE
  • Importation de liste des électeurs par le DGE
  • Gestion des candidats se présentant à l'élection
    • Ajout des candidats
    • Génération des codes d'authentification des candidats
    • Suivi de l'état d'avancement des parrainages

Candidats

  • Recevoir son code d'authentification par mail pour pouvoir se connecter
  • Suivre l'évolution journalière de son parrainage

Électeurs

  • Créer un compte pour pouvoir parrainer
  • Recevoir code de vérification pour pouvoir créer le compte
  • Parrainer un candidat

Défis et Solutions

Les principaux défis rencontrés lors du projet incluaient la coordination efficace entre les membres de l'équipe, l'intégration fluide du front-end et du back-end, ainsi que la gestion des données et de la sécurité. Pour y remédier, nous avons adopté une méthodologie agile avec des réunions régulières afin d'assurer une bonne communication et un suivi rigoureux des tâches. L'architecture MVC de Laravel a facilité l'organisation du code et l'ORM Eloquent a simplifié la gestion des bases de données. Enfin, des tests rigoureux et un déploiement sur Always Data ont permis d'assurer la stabilité et l'accessibilité de l'application.
